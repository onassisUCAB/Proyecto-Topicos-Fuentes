### Instrucciones para ejecutar

1. Clonar el repositorio de github
2. Abrir la terminal git-bash
3. Hacer cd en hasta estar ubicado en la carpeta dist dentro de ucab-tasks, puede copiar y pegar:
`cd ucab-tasks/dist`
4. Ejecutar `node main` (con el puerto 3000 libre)
5. Presionar ctrl+click en el link que aparece en la terminal (en caso de alg√∫n fallo correr npm install en la carpeta ucab-tasks y volver al paso 3).

### Instrucciones para hacer pruebas unitarias
1. Clonar el repositorio de github
2. Abrir la terminal git-bash
3. Hacer cd en hasta estar ubicado en la carpeta ucab-tasks, puede copiar y pegar:
`cd ucab-tasks`
4. Ejecutar `npm run test`
5. Esperar los resultados.

No es necesario ejecutar o instalar nada adicional, en caso de que no funcionen las instrucciones contactar a los desarrolladores.
